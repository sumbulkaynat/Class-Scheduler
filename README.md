# Scheduler Web Application


## Backend

### Project setup
```
npm install
```

### Run
```
node server.js
```

## Frontend
### Set port
.env
```
PORT=8081
```
### Project setup

In the project directory, you can run:

```
npm install
# or
yarn install
```

or

### Compiles and hot-reloads for development

```
npm start
# or
yarn start
```

Open [http://localhost:8081](http://localhost:8081) to view it in the browser.
